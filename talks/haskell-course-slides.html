1: The Language

ghci*


$ ghci

Prelude>
*Techically a tool, but you need it.

Expressions


$ ghci

Prelude> 5 + 5
10

Prelude> let x = "hello, "

Prelude> let y = "world"

Prelude> x ++ y
"hello, world"

Getting Types


Prelude> :t "hello"
"hello" :: [Char]

Prelude> let x = 7 :: Int

Prelude> :t x + x
x + x :: Int

Prelude> :t id
id :: a -> a

Prelude> :t (++)
(++) :: [a] -> [a] -> [a]

Other helpful ghci stuff


*Prelude> :info id
id :: a -> a    -- Defined in ‘GHC.Base’

*Prelude> :help
... lots of stuff ...

Built in Data Types

Bool


$ :info Bool
data Bool = False | True 	-- Defined in ‘ghc-prim:GHC.Types’
... 
*ghci> :t True
True :: Bool
*ghci> :t False
True :: Bool

Bool Functions


*ghci> :t (&&)
(&&) :: Bool -> Bool -> Bool	
*ghci> True && False 
False

*ghci> :t (||)
(||) :: Bool -> Bool -> Bool	
*ghci> True && False 
True

*ghci> :t not
not :: Bool -> Bool
*ghci> not True
False

Another useful ghci command

List Functions


*ghci> 1 : [2,3] -- append element to front of list
[1,2,3]

*ghci> [True, True] ++ [False, False] -- append two lists
[True,True,False,False]

*ghci> head [1,2,3]
1

*ghci> head []
*** Exception: Prelude.head: empty list

*ghci> tail [1,2,3]
[2,3]

*ghci> tail []
*** Exception: Prelude.tail: empty list

More List Functions


*ghci> length [True, False, True]
3

*ghci> ['a', 'b', 'c', 'd'] !! 0
'a'

*ghci> ['a', 'b', 'c', 'd'] !! 2
'c'

*ghci> map (\x -> x + 10) [1,2,3,4]
[11,12,13,14]

*ghci> filter even [1..10]
[2,4,6,8,10]

Strings

A String is just a [Char] 

*ghci> :browse Data.String
...
type String = [Char]
...

*ghci> :t "hello, world"
"hello, world" :: [Char]

*ghci> ['a'..'z'] ++ ['z','y'..'a']
"abcdefghijklmnopqrstuvwxyzzyxwvutsrqponmlkjihgfedcba"

*ghci> words "Hello, my name is Josh."
["Hello,","my","name","is","Josh."]

Tuples

Heterogeneous Product type 

*ghci> ("Hello",True)
("Hello",True)

*ghci> :t ("Hello",True)
("Hello",True) :: ([Char], Bool)

*ghci> :t ("Hello",True,'c')
("Hello",True,'c') :: ([Char], Bool, Char)

*ghci> :t (,)
(,) :: a -> b -> (a, b)

*ghci> :t (,,)
(,,) :: a -> b -> c -> (a, b, c)

Unit: The Empty Tuple

Represents the lack of data. 

*ghci> ()
()

*ghci> :t ()
() :: ()

*ghci> :info Maybe

*ghci> :t Just

*ghci> Just 5

*ghci> Just "hello"

3: Tools

Cabal: Haskell build system

  • Configure sources
  • Build libraries
  • Run tests
  • Build executables
  • ... basically everything ...

Clone and build the course repo


$ git clone https://github.com/joshcough/HaskellCourse.git

$ cd HaskellCourse

$ cabal sandbox init

$ cabal install --enable-tests --only-dependencies

$ cabal build

*Please raise your hand it this doesn't work!!*
